概述
-------------------------------------------------------------------------------

Leeln Repository 是一个java与数据交互层，常规中我们有人称为“DAO”。这部分功能通常是提供与数据库交互的接口。

这个项目带有一定的研究性质，如果你想了解请访问https://github.com/fightteam。

Leeln Repository 采用 Apache 2.0 许可. 详细请看license.txt。
